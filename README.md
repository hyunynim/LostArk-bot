# Discord bot for LostArk

# Command List(start with '!')
- 하위
- 강화 [확률]
- 주사위
- 원정대등록 [캐릭터 명]
  - 원정대 등록 시, 원정대에 포함되어 있는 캐릭터 중, 1325 레벨 이상의 모든 캐릭터 등록
- 원정대 [캐릭터 명] [주간|일일]
  - 해당 캐릭터의 주간/일일 컨텐츠 목록 확인
- 캐릭터추가 [캐릭터 명]
- 마리 [크리스탈 가격]
- 항아리
  - 성공 확률 10%
